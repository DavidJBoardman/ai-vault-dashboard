Screenshot Guidelines

Store screenshots in this folder using a simple structure based on the documentation flow.

Folder layout

- overview/
- installation/
- step-1/
- step-2/
- step-3/
- step-4/
- step-5/
- step-6/
- step-7/
- step-8/

Naming

Use short, descriptive file names.

Examples:

- upload-screen.png
- projection-preview.png
- segmentation-mask-panel.png
- trace-source-selector.png

Recommended usage in Markdown

![Projection preview](../images/step-2/projection-preview.png)

Place each screenshot directly after the instruction it supports.

Guidance

- prefer several focused screenshots over one very large image
- use annotations only when they clarify a specific action
- keep alt text meaningful
- crop screenshots to the relevant interface area when possible
