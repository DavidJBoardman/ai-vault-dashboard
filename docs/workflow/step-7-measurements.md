# Step 7: Measurements and Analysis

## Purpose

This step computes quantitative results from the selected trace geometry.

## What you do here

- inspect available ribs or trace lines
- run measurement calculations
- review derived values such as radius, rib length, apex, and springing points
- compare or group rib results where relevant

![Measurements workspace](../images/step-7/measurements-workspace.png)

## Expected result

Before moving on, you should have:

- the measurements needed for interpretation
- a reviewed set of geometric results associated with the traces
