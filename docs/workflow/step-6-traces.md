# Step 6: 3D Geometry Traces

## Purpose

This step manages trace geometry in 3D.

## What you do here

- inspect automatically detected intrados lines
- optionally import manual traces from a `3DM` file
- compare automatic and manual sources
- choose which trace set should be used downstream

## Trace sources

The software supports:

- automatic traces derived from prior analysis
- manual traces imported from Rhino `3DM`
- combined review of both

![Trace source selection](../images/step-6/trace-source-selection.png)

## Expected result

Before moving on, you should have:

- a confirmed set of 3D traces for measurement and interpretation
