# Step 5: Reprojection to 3D

## Purpose

This step maps selected analysis results back into the 3D point cloud.

## What you do here

- choose which segmentation groups to include
- preview the reprojection result in 3D
- inspect masked and unmasked points
- prepare intrados tracing inputs where needed

## Interface emphasis

This step is about checking whether the selected 2D interpretation still makes sense when viewed again in 3D space.

![3D reprojection preview](../images/step-5/reprojection-preview.png)

## Expected result

Before moving on, you should have:

- a usable 3D reprojection preview
- confidence that the selected features and exclusions are appropriate
