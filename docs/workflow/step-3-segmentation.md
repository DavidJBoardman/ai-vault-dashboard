# Step 3: Segmentation

## Purpose

This step identifies and organises relevant architectural features on the projection images.

## What you do here

- choose a projection image
- work with segmentation prompts
- define areas of interest using polygons, boxes, or ROI tools
- review masks and visibility
- organise or save segmentation results

## Important interaction modes

The interface includes:

- polygon prompting
- box prompting
- ROI definition
- mask visibility and editing controls

![Segmentation workspace](../images/step-3/segmentation-workspace.png)

## Expected result

Before moving on, you should have:

- segmentation masks that isolate the relevant architectural elements
- a saved segmentation state suitable for geometric interpretation
