# Step 1: Upload E57 Scan

## Purpose

This step brings the point-cloud dataset into the application and prepares the initial preview.

## What you do here

- choose an `E57` file from disk
- upload or open the file in the desktop application
- inspect the point-cloud preview
- confirm that the dataset loaded correctly

## Main controls in the interface

- file selection for `E57` input
- drag-and-drop upload support
- point-cloud preview
- point-count controls for preview density
- backend health indicator

![Upload screen](../images/step-1/upload-screen.png)

## Expected result

Before moving on, you should have:

- a successfully loaded scan
- a visible point-cloud preview
- basic confidence that the dataset is valid and usable
