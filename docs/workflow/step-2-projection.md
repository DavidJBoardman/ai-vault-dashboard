# Step 2: Generate Projection

## Purpose

This step converts the loaded 3D scan into 2D projection views that can be used in later analysis.

## What you do here

- choose a projection perspective
- adjust resolution or display settings
- review the generated 2D projection
- confirm the projection is suitable for segmentation and geometry work

## Typical perspectives

The interface supports multiple views, including top, bottom, and directional perspectives such as north, south, east, and west.

![Projection settings and preview](../images/step-2/projection-preview.png)

## Expected result

Before moving on, you should have:

- at least one usable 2D projection
- a view that clearly exposes the features needed for the next steps
