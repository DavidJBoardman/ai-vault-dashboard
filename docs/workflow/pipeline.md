# Workflow Overview

The application is organised as an 8-step workflow. Each chapter in this section corresponds to one step in the software.

## Workflow sequence

1. Upload E57 scan
2. Generate 2D projection
3. Segmentation
4. 2D geometry analysis
5. Reprojection to 3D
6. 3D geometry traces
7. Measurements and analysis
8. Three-circle chord method analysis

## How to use this section

Work through the steps in order. Later stages depend on outputs from earlier ones, especially projection, segmentation, and geometry analysis.
