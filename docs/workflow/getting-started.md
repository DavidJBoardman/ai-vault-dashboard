# Getting Started

Before you reach the 8-step workflow, you need to either create a new project or reopen one you saved earlier. Both options are available on the home screen when the application launches.

## Creating a new project

1. On the home screen, click the **New Project** card.
2. A name field appears below the cards. Type a descriptive name for the vault or survey session (for example, `Durham Cathedral — South Choir Aisle`).
3. Press **Enter** or click **Create Project**.

The application creates the project and takes you directly to Step 1 (Upload E57 Scan).

!!! tip
    Use a name that identifies the site, bay, and date so projects are easy to tell apart in the project list later.

## Opening an existing project

Saved projects are listed automatically inside the **Open Project** card on the home screen.

1. Find the project you want in the list. Each entry shows the project name, the date it was last modified, and the number of saved segmentation masks.
2. Click the row to open the project.

The application restores your saved state and returns you to the last step you were working on.

!!! note
    The project list is populated by the backend process. If you see **"Backend offline"** instead of your projects, wait a few seconds for the backend to finish starting up, then the list will appear automatically.

## Saving your work

Use the **Save** button in the top toolbar at any point during the workflow, or press `Ctrl+S` (Windows / Linux) or `Cmd+S` (Mac). It is good practice to save after completing each step, especially after segmentation and geometry edits.

## Deleting a project

Hover over a project row in the **Open Project** list and click the bin icon that appears on the right. A confirmation prompt prevents accidental deletion.
