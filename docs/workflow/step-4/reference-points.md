# Step 4B: Reference Points

## Purpose

This sub-stage prepares the reference points needed for the later matching and reconstruction work.

## What you do here

- inspect the current ROI and evidence layers
- place or review the reference points used by the analysis
- adjust points where needed
- save the point set before moving on

## Why it matters

The quality of the downstream matching depends on the reference-point setup. Inaccurate or incomplete points weaken the later typology and reconstruction outputs.

## Expected result

Before continuing within Step 4, you should have:

- a saved set of reference points
- point placement that is consistent with the visible geometric evidence
